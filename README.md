# 📊 Telecom Customer Behaviour & Support Performance — Tableau Dashboard

## 📌 Project Overview

This project is an interactive **Tableau Business Intelligence Dashboard** designed to analyze telecom customer behaviour, service usage, customer satisfaction, and support performance.

The dashboard converts raw telecom data into meaningful insights that help understand customer needs, improve service quality, and optimize support operations.

---

## 🎯 Business Objectives

The main objectives of this project are:

- Analyze customer behaviour and usage patterns
- Understand customer satisfaction levels
- Track telecom support performance
- Identify customer churn trends
- Evaluate service quality and response efficiency
- Support data-driven business decisions

---

## 📈 Key Dashboard Insights

### 👥 Customer Analysis
- Total customers
- Customer demographics
- Customer segmentation
- Service adoption patterns

### 📞 Support Performance Analysis
- Total support tickets
- Ticket resolution rate
- Average response time
- Support agent performance
- Issue category analysis

### 📉 Customer Behaviour Analysis
- Usage trends
- Plan preferences
- Customer satisfaction score
- Churn analysis
